PHASE 4: OPTIONS TRANSLATION TOURNAMENT
For every underlying trade generated in Phase 3, create an options translation replay.

DEFAULT OPTIONS TRANSLATION RULE
At each underlying trade entry timestamp:
- find the nearest valid 2 DTE option chain for that symbol
- select the 1-strike OTM contract in the trade direction
  - bullish trade -> call
  - bearish trade -> put
- use minute-level options bars if available
- attach greeks if available
- if exact 2 DTE or 1-strike OTM is unavailable, choose the nearest valid contract and record the fallback explicitly
- do not use future knowledge when selecting the contract

At each underlying trade exit timestamp:
- exit the option position at the first valid marketable bar after the exit signal
- stay RTH only

Important:
- Preserve timestamp alignment exactly
- Record the actual contract selected, including OCC symbol if available
- If options data is missing at a timestamp, mark the trade unresolved and do not fabricate fills
- If a strategy issues too many overlapping signals for a $25k account, enforce capital and position limits explicitly
